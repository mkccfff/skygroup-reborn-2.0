# Как самостоятельно менять тексты и изображения

Основной контент пока находится в файле `src/App.tsx`. Открывать проект удобнее всего в VS Code.

## Проекты

Найдите:

```ts
const OBJECTS = [
```

У каждого проекта можно изменить:

- `name` — название;
- `type` — тип проекта;
- `loc` — местоположение;
- `img` — картинка;
- `tags` — короткие преимущества;
- `price` — цена;
- `desc` — описание;
- `stats` — цифры в модальном окне.

Картинки проектов лежат в `public/assets`.

## Команда

Найдите:

```ts
const TEAM = [
```

Фотографии команды находятся в `public/assets/team`.

## Новости

Найдите:

```ts
const NEWS = [
```

Можно менять `tag`, `date`, `title`, `excerpt` и `img`.

## Цифры второго экрана

Найдите:

```ts
const DEV_STATS = [
```

`to` — число, `suff` — символ после числа, `l` — подпись.

## Награды

Найдите:

```ts
const AWARD_BADGES = [
```

Изображения наград лежат в `public/assets/awards`.

## Галерея

Найдите:

```ts
const GALLERY_IMAGES = [
```

Чтобы добавить изображение:

1. Положите оптимизированный JPG в `public/assets/gallery`.
2. Добавьте строку:

```ts
{ src: "assets/gallery/имя-файла.jpg", cap: "ПРОЕКТ · подпись" },
```

Рекомендуемый размер: 1920x1080, вес до 1 МБ.

## Фон второго экрана

Текущий файл:

```text
public/assets/developer-warm.jpg
```

Можно заменить его другим JPG с тем же именем. Лучше использовать горизонтальный кадр 16:9 размером около 1920x1080.

## Главное видео

Файлы первого экрана:

```text
public/assets/hero.mp4
public/assets/hero.jpg
```

`hero.jpg` показывается до загрузки видео.

## Общие правила для изображений

- Используйте латинские имена файлов без пробелов.
- JPG подходит для фотографий и рендеров.
- PNG нужен для логотипов, дипломов и прозрачности.
- Не заменяйте файл тяжёлым оригиналом на 20-50 МБ.

## После любых изменений

Проверьте сборку:

```powershell
cd D:\codex\claude\skygroup-reborn-2.0
npm.cmd run build
```

Если команда завершилась без ошибки, сайт готов к публикации.
